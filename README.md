# AgentEarth.ai Website

This repository contains the source code and project assets for the AgentEarth.ai company website.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The repository includes a GitHub Pages workflow at `.github/workflows/deploy.yml`. Every push to `main` builds the Vite frontend and publishes the `dist` artifact to GitHub Pages.
